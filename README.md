# comments-without-context
See humanity at it's best/worst. Takes Facebook comments from various news sources and displays them out of the context of the article.

# Running

You need the following env vars: 
* `APP_ID` - Facebook app ID
* `APP_SECRET` - Facebook app secret
* `ACCOUNTS` (optional) - comma separated list of FB account usernames to take the comments from

To run:  `npm start`
